# Sales-Scoreboard-Dashboard
-Designed and developed an interactive Sales Scoreboard Dashboard in Power BI to analyze sales, profit, and YoY performance across multiple regions, products, and customer segments.
-Built dynamic KPI cards using SWITCH-based DAX measures, enabling users to seamlessly toggle between Sales and Profit metrics, improving metric comparison efficiency.
-Implemented drill‑through functionality from regional performance charts to detailed Product and Customer pages, allowing stakeholders to explore insights at granular levels and identify drivers of performance.
Created dynamic tooltips that displayed contextual insights—such as YoY trend summaries and sales distribution—enhancing data storytelling and user engagement.
Added interactive buttons and bookmark-driven navigation for smooth movement between Overview, Product Details, Category Details, and Customer Details pages.
Designed time‑series visualizations for Sales vs Profit trends, empowering leadership to identify seasonality patterns and anomalies.
Built a region-wise Sales and Profit analysis using treemaps and bar charts, enabling comparison of regional performance and growth opportunities.
Structured an optimized data model supporting efficient calculations, reduced refresh time, and improved report responsiveness.
Ensured a clean, professional dark-themed UI/UX promoting readability and executive‑level presentation quality.
Delivered a comprehensive, self-service dashboard that enabled consistent, data-driven decision-making for business stakeholders.
